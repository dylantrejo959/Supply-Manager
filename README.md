# Sistema de Control de Insumos y Abastecimiento

## Descripción
Sistema desarrollado en C# Windows Forms para gestionar insumos,
movimientos de inventario y control de abastecimiento.

## Tecnologías
- C#
- Windows Forms
- SQL Server
- ADO.NET
- GitHub

## Arquitectura
Arquitectura en capas:
- Forms
- Services
- Data
- Entities
- Utils

## Roles
- Administrador
- Almacenista
- Supervisor
